# 📸 Kochi Lens E-commerce System

## 1. Bối cảnh dự án

Công ty **Kochi Lens** muốn xây dựng một website bán thiết bị ngành ảnh.  
Hệ thống cho phép khách hàng đặt hàng trực tuyến, đồng thời dữ liệu đơn hàng được đồng bộ về BackEnd để:

- 📦 Bộ phận kho đóng gói hàng
- 🧾 Bộ phận kế toán xuất hóa đơn

---

## 2. Chức năng chính

### 2.1. Quản lý Sản phẩm (PIM)
- Quản lý biến thể (Màu sắc, kích thước)
- Hiển thị tồn kho realtime

### 2.2. Giỏ hàng & Thanh toán (SC&CO)
- Tính toán thuế, phí vận chuyển
- Tích hợp cổng thanh toán (VNPay, MoMo)

### 2.3. Quản lý Khách hàng (CRM)
- Phân loại:
  - Guest (Khách vãng lai)
  - B2C Member
  - B2B Member
